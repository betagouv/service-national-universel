Windows

  1. Assurez-vous que votre imprimante est correctement installée et configurée sur votre ordinateur.
  2. Localisez et ouvrez le dossier contenant les documents PDF que vous souhaitez imprimer.
  3. Sélectionnez les fichiers que vous souhaitez imprimer.
  4. Cliquez avec le bouton droit de la souris sur l'un des documents PDF sélectionnés, puis sélectionnez "Imprimer" dans le menu qui apparaît.
  5. Dans la fenêtre d'impression qui s'ouvre, vérifiez les paramètres d'impression tels que l'imprimante sélectionnée, le nombre de copies, l'orientation, etc. et ajustez-les selon vos besoins.
  6. Cliquez sur le bouton "Imprimer" pour lancer l'impression des documents PDF sélectionnés.
  7. Attendez que l'impression soit terminée, puis vérifiez les documents pour vous assurer que tout s'est correctement déroulé.

Mac OS

  1. Assurez-vous d'avoir l'application Aperçu installée sur votre ordinateur. Si ce n'est pas le cas, vous pouvez la télécharger et l'installer depuis le site officiel d'Apple.
  2. Localisez le premier fichier PDF que vous souhaitez ouvrir dans Aperçu. Ceci :
    1. Ceci peut être fait en naviguant vers le dossier où se trouve le fichier à l'aide du Finder
    2. Cliquez avec le bouton droit sur le fichier PDF
    3. Sélectionnez "Ouvrir avec" dans le menu contextuel, puis choisissez Aperçu.
  3. Une fois que le fichier PDF est ouvert dans Aperçu, vous verrez une barre latérale à gauche de la fenêtre.
  4. Assurez-vous que cette barre latérale est visible. Si elle ne l'est pas, vous pouvez l'afficher en cliquant sur l'icône de la barre latérale dans la barre d'outils d'Aperçu.
  5. Maintenant, localisez les autres fichiers PDF que vous souhaitez ajouter à votre document : naviguez vers le dossier où se trouvent ces fichiers, sélectionnez les et faites-les glisser vers la barre latérale d'Aperçu comme sur la capture ci-jointe. Vous verrez que chaque fichier PDF ajouté s'affiche sous forme de miniature dans la barre latérale.
  6. Une fois que vous avez ajouté tous les fichiers PDF souhaités à la barre latérale, vous pouvez réorganiser leur ordre si nécessaire. Il suffit de faire glisser les miniatures vers le haut ou vers le bas pour les réarranger.
  7. Maintenant que vous avez combiné tous les fichiers PDF en un seul document dans Aperçu, vous pouvez imprimer le fichier obtenu.
  8. Pour cela, cliquez sur le menu "Fichier" dans la barre de menu d'Aperçu, puis sélectionnez "Imprimer". Une boîte de dialogue d'impression s'ouvrira.
  9. Dans la boîte de dialogue d'impression, vous pouvez sélectionner l'imprimante appropriée et configurer les paramètres d'impression selon vos préférences. Vérifiez les options telles que la mise en page, l'orientation, la qualité d'impression, etc.
  10. Si vous souhaitez enregistrer le fichier combiné en tant que PDF unique avant de l'imprimer, assurez-vous de cocher l'option "Enregistrer au format PDF" ou "PDF" dans la boîte de dialogue d'impression. Vous pouvez choisir l'emplacement et le nom du fichier PDF résultant.
  11. Si vous ne souhaitez pas enregistrer le fichier combiné en tant que PDF unique, assurez-vous que l'option "Enregistrer au format PDF" ou "PDF" n'est pas cochée. Ensuite, cliquez simplement sur le bouton "Imprimer" dans la boîte de dialogue d'impression.
  12. Lorsque l'impression est terminée, vous pouvez quitter l'application Aperçu. Si vous avez choisi de ne pas enregistrer le fichier combiné en tant que PDF unique, une boîte de dialogue de confirmation apparaîtra pour vous demander si vous souhaitez enregistrer les modifications. Sélectionnez simplement "Ne pas enregistrer" pour fermer l'application sans enregistrer le fichier.
