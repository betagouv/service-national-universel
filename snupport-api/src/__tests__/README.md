# Tests SNUpport API

## Installation

Pour installer les dépendances de test :

```bash
npm install
```

## Exécution des tests

Pour exécuter tous les tests :

```bash
npm test
```

Pour exécuter les tests en mode watch :

```bash
npm run test:watch
```
